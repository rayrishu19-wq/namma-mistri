# Namma-Mistri (ನಮ್ಮ ಮಿಸ್ತ್ರಿ) - Construction Assistant

### 👷‍♂️ Project Overview
**Namma-Mistri** is a GenAI-powered mobile assistant designed for local builders (Mistris) and masons in rural and semi-urban India. The app solves the critical problem of material wastage and inefficient site management by providing real-time calculations and AI-driven professional advice.

### 🚀 Key Features
1. **Material Calculator**: Precise estimation of bricks, cement bags, and sand loads based on wall dimensions.
2. **Multi-Site Management**: Ability to manage multiple construction projects simultaneously with separate records.
3. **Labor Diary**: A digital logbook for tracking worker attendance and advance payments, with automatic balance calculation.
4. **Site Progress Gallery**: A visual log of work progress using the device camera.
5. **Mistri AI**: A bilingual (English/Kannada) AI assistant powered by **Google Gemini 2.5 Flash**, providing expert construction advice.

### 🛠️ Technical Stack
- **Frontend**: HTML5, CSS3 (Rugged Mobile-First Design).
- **Logic**: Vanilla JavaScript (ES6+).
- **Database**: Browser LocalStorage (for offline data persistence).
- **AI Integration**: Google Gemini API (v1 REST Endpoint).
- **Deployment**: Designed to be wrapped as a native Android APK using Capacitor.

### 📐 Engineering Formulas Used
- **Brickwork Volume**: `Length * Height * Thickness`.
- **Mortar Estimation**: 1:6 ratio for cement-sand mixture.
- **Wastage Factor**: 5% standard allowance included in calculations.

### 🌍 Accessibility
- **Bilingual UI**: Both English and Kannada support to bridge the language gap.
- **Rugged UI**: High-contrast colors and oversized interactive elements for use in outdoor, high-glare, and dusty environments.
